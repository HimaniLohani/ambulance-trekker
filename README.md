# 🚑 Ambulance Trekker

A full-stack web application that enables users to **book ambulances quickly and efficiently** during emergencies. The system provides a smooth interface for requesting ambulance services and managing bookings.

---

## 📌 Features

* 🚑 Book ambulance in real-time
* 👤 User authentication (Signup/Login)
* 📋 View booking details
* 🔄 Track booking status
* 🖥️ Responsive and user-friendly UI

---

## 🛠️ Tech Stack

### Frontend

* React.js
* CSS

### Backend

* Node.js
* Express.js

### Database

* MongoDB

## 🚀 Getting Started

### 🔧 Prerequisites

Make sure you have installed:

* Node.js
* npm or yarn
* MongoDB

---

### ▶️ Run Backend

```
cd backend
npm install
npm start
```

---

### ▶️ Run Frontend

```
cd frontend
npm install
npm start
```
